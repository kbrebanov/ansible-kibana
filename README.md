kibana
======

[![Build Status](https://travis-ci.org/kbrebanov/ansible-kibana.svg?branch=master)](https://travis-ci.org/kbrebanov/ansible-kibana)

Installs and configures Kibana

Requirements
------------

This role requires Ansible 1.9 or higher.
